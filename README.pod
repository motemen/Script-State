=head1 NAME

Script::State - Keep script local variables

=head1 SYNOPSIS

  use Script::State;

  script_state my $count = 0;

  say $count++;

=head1 DESCRIPTION

Script::State stores local variables to file after script execution.

Try running

  perl -Ilib t/fib.pl; echo
